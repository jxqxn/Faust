# 《密教模拟器》的组合式叙事引擎：被《苏丹的游戏》保留的五个体验原子

> 文档状态：设计分析
>
> 更新日期：2026-07-16
>
> 证据边界：本文不是《苏丹的游戏》开发团队的历史自述，也不声称还原其实际设计流程。本文根据两作的最终体验与系统结构，提出一套用于分析、继承或改造同类游戏的设计判断。

## 1. 结论

《苏丹的游戏》删改了《密教模拟器》的许多形式：抽象动词、神秘学概念、实时多线程压力，以及完全自主的目标寻找。

但它保留了更深层的交互语法：**玩家通过卡牌的组合，发现并改变一个持续运作的叙事系统。**

这不是单一的“卡牌玩法”，而是五个不能随意拆开的体验原子：

1. 桌面卡牌作为统一的玩法语言；
2. 将卡牌投入某处来产生意义的组合机制；
3. 有限揭示下的发现；
4. 会持续累积的状态；
5. 由系统组合生成的叙事。

若删除其中任何一项，作品仍然可以拥有任务、角色、压力与分支，但不再继承《密教模拟器》最核心的组合式叙事引擎。

## 2. 五个体验原子

### 2.1 桌面卡牌作为统一的玩法语言

人物、物品、知识、状态、机会与任务，都以可移动、可消耗、可观察的卡牌存在。玩家不是在菜单中选择一个预制动作，而是在桌面上组织世界。

这层统一语言让叙事信息与系统资源使用同一种媒介呈现：一张卡既可以是故事中的某个人，也可以是可被投入、保留、失去或转化的游戏对象。

### 2.2 “把卡放进某处”来产生意义的组合机制

核心行动是把合适对象投入行动、事件或槽位；条件命中后，系统给出结果。它将叙事推进转化为具体的资源编排与语义匹配。

玩家的关键问题不是“下一段剧情是什么”，而是：

- 我该把什么投入这里？
- 这个对象是否满足隐藏或显式条件？
- 为了眼前结果，我愿意消耗、占用或暴露什么？

### 2.3 有限揭示下的发现

玩家不会从一开始就知道全部配方、分支与后果，必须通过尝试、线索与失败逐渐建立理解。

《苏丹的游戏》没有放弃发现，而是降低了纯盲猜的比例：仪式槽位、条件提示、等级与地图入口将探索收束为较可读的局部问题。保留的关键不是故意晦涩，而是让玩家仍须提出假设、验证假设，并获得“我终于理解了这个系统”的发现感。

### 2.4 会累积的状态

一次选择不只是结算一笔资源，而会留下标签、损失、关系、装备、机会或剧情条件；未来内容会读取这些痕迹。

没有状态累积，卡牌组合会退化为一组彼此无关的一次性谜题。状态的持续性使玩家能够把过去的行动理解为世界历史，也使“这次怎么做”成为对未来可选项的投资或破坏。

### 2.5 由系统组合生成叙事

故事不完全预写成单线剧情，而是从“谁参与了什么、消耗了谁、留下什么状态、之后又抽到什么”中生长出来。

这使玩家可以复述一段属于自己的经历，而不只是复述编剧安排的固定情节。它也是两作最深的共同点：系统的组合结果不仅改变数值，还产出具有记忆点的叙事。

## 3. 核心循环

```text
获得卡牌或机会
→ 组合并投入行动
→ 命中条件
→ 改变状态与关系
→ 开启新的机会与故事
```

五个体验原子共同构成这个循环：卡牌提供可操作对象，组合产生行动，有限信息提供发现，状态累积保存结果，涌现叙事赋予结果意义。

## 4. 《苏丹的游戏》改了什么，没有改什么

《苏丹的游戏》主要改写的是这条循环的**驱动**与**对象**，而不是循环本身。

| 层级 | 《密教模拟器》倾向 | 《苏丹的游戏》的改造 |
|---|---|---|
| 目标来源 | 玩家自主寻找目标 | 外部权力提出不可拒绝的任务 |
| 时间结构 | 实时、多线程计时 | 离散日期与集中期限 |
| 组合对象 | 知识、概念、工艺与资源 | 具体人物、关系、身份、物品与经历 |
| 结果承载 | 资源与进程为主 | 人物状态、关系与政治后果 |

因此，最短的对比是：

> 《密教模拟器》让玩家在组合中摸索神秘知识；《苏丹的游戏》让玩家调度具体的人，回应权力施加的问题。

前者的开放炼金台被改造成后者的权力调度台，但两者都依赖同一套组合式叙事引擎。

## 5. 对原创设计的约束

原创作品不应直接复制苏丹卡、宫廷人物、仪式名称、骰子、日历节奏或四类任务。它可以替换题材、压力来源、可调度对象与时间结构。

但若希望继承这一类游戏的核心体验，应逐项确认：

- 世界中的关键对象是否以统一、可观察、可操作的形式出现？
- 玩家是否通过组合对象，而不是只在菜单中选择文本选项，来推动叙事？
- 玩家是否需要在有限信息下发现系统关系，而非机械执行已知配方？
- 每次选择是否留下会被后续读取的状态？
- 组合与状态能否共同产生可复述、可记忆的个人经历？

这些问题全部回答“是”，才说明作品保留了组合式叙事引擎；之后才应讨论如何用新的压力、对象与主题，让它成为不同于《密教模拟器》或《苏丹的游戏》的原创作品。
